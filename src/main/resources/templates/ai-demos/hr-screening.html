HR Resume Screening

AI-powered hiring automation using Amazon Textract and Comprehend for intelligent resume processing

About This Demo

This AI-powered HR screening system demonstrates enterprise-grade document processing capabilities using AWS machine learning services. The application automatically extracts, analyzes, and matches candidate resumes with job requirements.

Built with modern Java Spring Boot architecture and integrated with AWS Textract for document OCR and AWS Comprehend for natural language processing.

Key Features

  • Automatic resume parsing and text extraction
  • Skills and experience identification
  • Intelligent candidate matching
  • Real-time processing and ranking
